# Project Context (Maintained by Orchestrator)

## Project
SalesForge CRM — FastAPI backend + React/TypeScript frontend

## Tooling
- **Backend package manager**: pip / pyproject.toml
- **Backend test runner**: pytest (`cd backend && pytest`)
- **Backend coverage**: `pytest --cov --cov-fail-under=80`
- **Backend lint**: `cd backend && ruff check .`
- **Frontend package manager**: npm
- **Frontend test runner**: vitest (`cd frontend && npm run test`)
- **Frontend lint**: `cd frontend && npm run lint`
- **Coverage thresholds**: 80% lines/branches/functions/statements (`.coverage-thresholds.json`)
- **DB migrations**: alembic
